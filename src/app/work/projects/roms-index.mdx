## Project Overview

A visually stunning index of classic gaming consoles providing quick access to ROM archives. Features 23 consoles across Nintendo, Sony, Microsoft, and Sega platforms with high-quality console imagery and direct links to preservation databases.

## Console Library

- **Nintendo** (13 consoles)
  - NES, SNES, N64, GameCube, Wii, Wii U
  - Game Boy, GBC, GBA, DS, 3DS
  - Virtual Boy, Switch

- **Sony PlayStation** (6 consoles)
  - PlayStation 1, 2, 3, 4
  - PSP, PS Vita

- **Microsoft** (2 consoles)
  - Xbox, Xbox 360

- **Sega** (2 consoles)
  - Dreamcast, Saturn

## Features

- **Visual Design**
  - Purple gradient background
  - High-quality console artwork (v2.1 Dark Color pack)
  - Hover animations with scale effects
  - Fully responsive grid layout

- **User Experience**
  - Standalone full-page design
  - No header/footer for immersive experience
  - Direct links to Myrient archive
  - Fan translations section (TraduSquare)

- **Performance**
  - Lazy loading for all images
  - Next.js Image optimization
  - Responsive breakpoints (desktop, tablet, mobile)

## Technical Stack

- **Framework**: Next.js 15
- **Styling**: SCSS Modules
- **Images**: Optimized PNGs with lazy loading
- **Links**: Myrient, RomsLab, RomsFun
