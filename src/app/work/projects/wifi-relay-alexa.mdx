## Technical Specifications

- **Microcontroller**: ESP8266 (NodeMCU/Wemos D1)
- **Relay**: 5V relay module
- **Power**: USB or 5V adapter
- **Connection**: WiFi 2.4GHz
- **Protocol**: MQTT, HTTP
- **Integration**: Alexa Skills, Google Actions
